Before you deploy

  1. Edit config.js and set your Genesys Cloud clientId, environment, and GitHub Pages redirectUri.
  2. Create or update the OAuth client in Genesys Cloud so the redirect URI matches your GitHub Pages URL exactly.
  3. Make sure the signed-in user and OAuth client have the permissions/scopes needed for the API calls used by this checker.
